Use this tool to update the current todo list for the current session.

When to use this tool:
- Use this tool proactively and frequently to ensure that you are aware of the current task list and progress
- Use this tool as often as possible - the todo list should be kept up to date at all times

This tool takes no parameters. It returns a structured todo list with status and content for each todo item.

The todo list is shared between the user and assistant, so the assistant can see what tasks are pending, in progress, or completed.
